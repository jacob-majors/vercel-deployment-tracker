What it does

Everything in one place.

▲

Vercel Deployments

Track production and preview deployments in real time. See status, commit messages, and jump straight to the preview URL.

GitHub Actions

Monitor CI workflow runs across all your repos. Spot failures immediately and open the run directly from the panel.

◯

Always accessible

Lives quietly in your menu bar. Available as an Electron app and a native SwiftUI build — no Dock icon, no clutter.

⌥

Works out of the box

Starts in mock mode with polished sample data. Add tokens to .env whenever you are ready to go live.

Quick start

Up and running
in two minutes.

Clone the repo, drop in your API tokens, and run the dev command. Deplog starts in mock mode with no tokens required.

VERCEL_TOKEN Vercel access token
GITHUB_TOKEN GitHub personal access token
VERCEL_TEAM_ID Optional — for team projects
GITHUB_REPOS Optional — defaults to all repos 
# Clone the repo
$ git clone https://github.com/jacob-majors/vercel-deployment-tracker
$ cd vercel-deployment-tracker

# Add your tokens
$ cp .env.example .env
# Edit .env with VERCEL_TOKEN and GITHUB_TOKEN

# Install and run
$ npm install
$ npm run dev

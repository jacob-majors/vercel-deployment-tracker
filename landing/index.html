What Vigil monitors.

Vercel

Deployment status

Production and preview deployments, updated in real time. See the commit, branch, and preview URL at a glance.

GitHub

Workflow runs

CI status across all your repositories. Jump directly to the run on GitHub from inside the panel.

Interface

Menu bar native

Available as an Electron app and a native SwiftUI build. No Dock icon, no extra windows.

Setup

Mock mode included

Works immediately with sample data. Add tokens to .env to switch to live data.

Get started.

Clone the repo and add your API tokens. Vigil runs in mock mode until tokens are present, so there are no blank states on first launch.

VERCEL_TOKEN Vercel access token
GITHUB_TOKEN GitHub personal access token
VERCEL_TEAM_ID Optional — for team projects
GITHUB_REPOS Optional — auto-discovers all repos
bash 
# clone
$ git clone https://github.com/jacob-majors/vercel-track-macos
$ cd vercel-deployment-tracker

# configure
$ cp .env.example .env

# run
$ npm install && npm run dev
